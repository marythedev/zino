# Zino | Shop Online

This project is a powerful **e-commerce platform** designed specifically for retail business. It provides a smooth product showcase, encourages customer engagement and feedback, and makes online sales easy through a user-friendly web interface and an intuitive admin dashboard.

## Tech Stack
### MERN - MongoDB, Express, React, Node

| Technology | Role | Description |
|------------|------|-------------|
| **MongoDB** | Database | The database management system, providing a scalable and flexible NoSQL database to store and manage data efficiently. |
| **Express.js** | Backend Framework | Express is employed as the web application framework, facilitating the development of robust and scalable back-end APIs. |
| **React.js** | Frontend Library | The frontend's user interface is built with React.js, a powerful JavaScript library for creating dynamic and interactive user interfaces. |
| **Node.js** | Backend Server Runtime | The backend's server runtime environment is based on Node.js, allowing for the execution of server-side JavaScript code and ensuring high performance.. |

## Developing Environment

| Technology | Role | Description |
|------------|------|-------------|
| **Visual Studio Code** | IDE | Visual Studio Code serves as the primary integrated development environment (IDE) with extensions to support JavaScript, React and Node.js development. |
| **npm** | Package Manager | npm is utilized for package management and dependency tracking, streamlining the integration of third-party libraries and tools. |
| **Figma** | UI/UX | Figma is used to create prototypes of interface design and animations for the web store pages. |
| **Bootstrap 5 & CSS** | UI/UX | Bootstrap 5 is utilized for front-end styling, ensuring a responsive and visually appealing design across various devices and screen sizes. |
| **Postman** | QA | Postman is employed for testing HTTP requests, allowing for the seamless testing and validation of back-end APIs independently of the front-end development. |
| **Jest testing Suite** | QA | Jest provides fast and parallel test execution, helping to speed up the testing process. It is designed to work right out of the box with minimal configuration required. It has been used within the project at different stages to assist with testing (via simple automation) and ensure product behaviour. |
| **Vercel** | Hosting Provider | A cloud platform for developing, deploying and scaling web apps. Vercel serverless has been used to deploy both the front and back end implementations of the prototype design. |
